# AlgoMentor

AlgoMentor is an AI-powered Data Structures and Algorithms (DSA) learning platform designed to help students master coding problems. The platform guides students with progressive hints, performs in-depth code complexity analysis, outlines multiple approaches (from brute-force to optimal), and tracks their learning journey.

---

## Key Features (Planned)

- **Progressive Hints**: Learn by solving step-by-step instead of directly looking at the solution.
- **AI-Powered Code Analysis**: Get personalized feedback on code efficiency, edge cases, and bug fixes.
- **Approach Explorer**: Compare different algorithmic approaches with side-by-side time and space complexity analyses.
- **Student Progress Dashboard**: Track solved problems, concepts mastered, and overall growth.

---

## Planned Technology Stack

- **Frontend**: React.js / Vite, TailwindCSS (for responsive, modern UI layout)
- **Backend**: Node.js, Express.js (REST API layer)
- **Database**: MongoDB (for user profile and progress tracking data persistence)
- **AI Integration**: Google Gemini API (for code reviews and hints generation)
